# Overview

Powershell extensions for gitlab CLI.

Extends (and requires) [python gitlab](https://github.com/python-gitlab/python-gitlab#python-gitlab).

## Examples

### `Get-GitlabProject`

```powershell
Get-GitlabProject 'group/project'
```

```powershell
Get-GitlabProject 42
```
